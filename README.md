# Linux C Programming Helper

**Linux C Programming Helper** — это интерактивное справочное приложение для изучения языка программирования C в среде Linux. Приложение предоставляет структурированные обучающие материалы (главы), разделенные по темам, и удобный просмотр исходного кода с подсветкой синтаксиса.

## ✨ Особенности

*   **Структурированное обучение:** Главы разбиты на логические блоки (например, "Базовый C", "GTK 4").
*   **Современный интерфейс:** Использование библиотек **GTK 4** и **Libadwaita** обеспечивает нативный и эстетичный вид в современных дистрибутивах Linux.
*   **Подсветка синтаксиса:** Интеграция с `GtkSourceView5` для удобного чтения обучающего кода.
*   **Тематическое оформление:** Поддержка различных тем оформления для комфортного обучения.

## 🛠 Требования

Для сборки и работы приложения вам понадобятся:

*   **Компилятор C:** С поддержкой стандарта **C23** (например, GCC или Clang последних версий).
*   **Система сборки:** `CMake` (версии 3.16 или выше).
*   **Библиотеки разработки:**
    *   `gtk4`
    *   `libadwaita-1`
    *   `gtksourceview-5`
    *   `pkg-config`

## 🚀 Сборка и установка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Limemani/Programming-c-lang-helper-for-linux.git
    cd Programming-c-lang-helper-for-linux
    ```

2.  **Создайте директорию для сборки:**
    ```bash
    mkdir build
    cd build
    ```

3.  **Сконфигурируйте проект с помощью CMake:**
    ```bash
    cmake ..
    ```

4.  **Скомпилируйте приложение:**
    ```bash
    make
    ```

5.  **Запустите приложение:**
    ```bash
    ./Linux_C_Programming_Helper
    ```

## 📂 Структура проекта

*   `src/` — исходный код приложения.
*   `include/` — заголовочные файлы (`.h`).
*   `resources/` — графические ресурсы и CSS-стили.
*   `CMakeLists.txt` — файл конфигурации сборки.
